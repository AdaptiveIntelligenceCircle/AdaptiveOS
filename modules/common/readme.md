# modules/common

This module defines **shared primitives** used across AdaptiveOS.

## Design principles
- No decision logic
- No learning logic
- No ethical reasoning
- No authority

## Purpose
- Structural consistency
- Traceability
- Human auditability
- Distributed coherence (AIN)

## Anti-goals
- DO NOT put AI logic here
- DO NOT put policy here
- DO NOT bypass governance or HMN
